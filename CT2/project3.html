Circuit Defender

Genre: Reaction Game | Tags: Action, Fast-Paced, Arduino-Based | Release: 2025

About the Game

In the ArtCenter Making Lab, an Arduino board faces a critical Bug invasion. As the last system maintainer, you must eliminate the Bugs to defend the system components!

Gameplay

Objective: Eliminate Bugs quickly and protect critical system modules.
Mechanics:

  • Precision shooting at Bugs
  • Avoid hitting system components (costs life)
  • Increasing difficulty over time
